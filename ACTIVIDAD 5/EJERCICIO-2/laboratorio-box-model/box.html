📦 Laboratorio: Modelo de Cajas

Comparación visual de content, padding, border, margin y overflow

1. Content

Esta caja solo tiene ancho y alto definidos. Sin padding ni border. El contenido ocupa todo el espacio disponible.

2. Padding

Esta caja tiene padding generoso. Notá el espacio entre el borde y el texto. El contenido respira más.

3. Border

Esta caja tiene un borde grueso y de color diferente. El borde se suma al tamaño total si no usás box-sizing: border-box.

4. Margin

Esta caja tiene margin grande. Notá el espacio externo que la separa de las demás cajas del laboratorio.

5. Border Radius

Esta caja tiene border-radius. Las esquinas se redondean según el valor que se le asigne.

6. Overflow: visible

El contenido se desborda fuera de la caja sin ser cortado. Este es el comportamiento por defecto del navegador cuando el contenido supera el tamaño definido de la caja y no se controla con overflow.

7. Overflow: hidden

El contenido que se sale de la caja es cortado y no se muestra. Lo que supera el límite definido simplemente desaparece sin dejar rastro visible para el usuario.

8. Overflow: scroll

Aparece una barra de desplazamiento siempre, aunque el contenido no la necesite. Permite al usuario hacer scroll para ver todo el contenido dentro de la caja definida.

9. Overflow: auto

La barra de desplazamiento aparece solo cuando el contenido realmente lo necesita. Es la opción más inteligente y recomendada para la mayoría de los casos en producción.
